# Live-Motion-and-Experiment-Analyzer-
Real-time motion tracking and physics experiment analyzer using OpenCV, NumPy, and Python.


A Python + OpenCV project that tracks moving objects in real time and analyzes motion parameters like displacement, velocity, and acceleration.

## Features
- Real-time object tracking
- Trajectory plotting
- Velocity calculation
- Acceleration calculation
- Experiment motion analysis

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

## How to Run
pip install -r requirements.txt
python motion_and_experiment_analyzer.py
